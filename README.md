# Hardware

	- Intel Edison
	- Sparkfun Base Block


# Pre-requisites

	https://software.intel.com/en-us/flashing-firmware-on-your-intel-edison-board-linux

	-  sudo dnf install dnf-util
	-  sudo ./flashall

# Connect

	- sudo screen /dev/ttyUSB0 115200

# Wifi

	- configure_edison --wifi

# SSH

	- configure_edison --password
